# 🚨 Helmet Detection with Alarm (V2)

## 📌 Overview
This project detects helmets in videos and triggers an alarm sound when a person is detected without a helmet.

## 🚀 Features
- Helmet detection using YOLOv8
- Works on video input
- 🔊 Alarm system for no-helmet detection
- Real-time safety monitoring

## 🧠 Model
- Custom trained YOLOv8 model

## ⚙️ Tech Stack
- Python
- OpenCV
- YOLOv8
- Flask
- FFmpeg

## ▶️ How it Works
1. Upload video  
2. Model detects helmets  
3. If no helmet → alarm triggers
